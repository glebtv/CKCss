## CKCsss

This is an update of the CKCss plugin to work with CKEditor 4

## Usage
        
        config.extraPlugins = "...,CKCss";
        config.toolbar = 'CMS';
        config.toolbar_CMS =
            [
                [..., 'CKCss']
            ];

You might need to add a locale file for your language
